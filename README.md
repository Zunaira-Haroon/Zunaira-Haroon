### 🧑‍💻 About Me

I'm a Senior Mobile Application Engineer with **6+ years of experience** in React Native and cross-platform mobile development, including 3 years in frontend development. My primary stack is React Native, and I also build complete web applications end-to-end — frontend in **React JS**, backend in **Node.js + Express**, with **MongoDB** for data.

I own the full product lifecycle — requirement discovery, backlog grooming, sprint planning, architecture, and release — partnering closely with product, design, backend, and QA teams.
